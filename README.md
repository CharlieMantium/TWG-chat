# Chat

This is a chat made as a recruitment task for one of the companies for position of Junior Software Developer.

### Table of Contents

- [Stack](#stack)
- [Setup](#setup)
- [Credits](#credits)

### Stack

Chat is created with:

- React Native
- Apollo
- Styled Components

### Setup

To run this site locally, install it using npm:

```
$ cd ./TWG-chat
$ npm install
$ npm start
```
